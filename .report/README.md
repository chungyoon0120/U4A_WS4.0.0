## ✍️ 기록 방법

그날 작업 끝에 해당 월 파일(`daily/YYYY-MM.md`)의 **맨 위**에 아래 블록을 추가한다.
달이 바뀌면 `daily/`에 새 월 파일을 만든다. 잔여 수치는 측정한 날만 적고, 안 했으면 `—`.
수치가 바뀌면 위 **진척 추이** 표에도 한 행 추가한다.

```md
## YYYY-MM-DD (요일)
**한 일**
- …
**완료 ✅**: …
**잔여**: 팝업 ?? · 코어 ?? · 합계 ?? (전일 대비 ▼?)
**메모**: …
```
